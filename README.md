## PDF-Generator-System-in-Laravel-9

Enter values and get or print PDF output in desired size

**How To Run**

- cp .env.example .env
- composer install
- npm install
- npm run dev
- php artisan migrate:fresh --seed
- php artisan optimize:clear
- php artisan serve
- register
